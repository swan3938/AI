# 🔄 思维技巧 · 逆向思考（Inversion）

正面太难想的问题，反过来往往豁然开朗。芒格的名言：「我只想知道我会死在哪，这样我就永远不去那儿。」

正向问"怎么成功"，答案无穷无尽、无从下手；反向问"**怎样保证失败**"，答案具体、可数、好避开。把"追求好结果"翻译成"排除坏结果"，难题就变成了排除题。

🎯 今日任务（这是个真练习，请动笔）：
1. 写下一个你正想做好的目标（带好团队 / 练好身体 / 学好某技能 / 经营好一段关系）。
2. **彻底反过来**：列出"要把这件事搞得稀烂，我该怎么做？"——逼自己写满 **7 条**，越具体越好。
3. 把每条取反，就是一份"高优先级避坑清单"。
4. 圈出其中**今天就能避开**的 1～2 条，立刻执行。

你会发现：很多时候你不需要更多"该做的"，只需要停掉几件正在"搞砸它"的事。

📚 延伸：雅各比的 "invert, always invert"；芒格《穷查理宝典》。
