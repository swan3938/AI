# 🔄 系统思维 · 二阶效应（Second-Order Thinking）

「然后呢？」——这三个字能拉开普通人和高手的差距。

一阶思维只看眼前结果：糖好吃 → 吃糖。
二阶思维继续追问：然后呢？→ 血糖飙升、犯困、长期发胖。

很多看似聪明的决定，败在没问「然后呢」：
- 限价令 → （一阶）东西变便宜 →（二阶）供给减少、黑市出现、更买不到。
- 给团队加人赶进度 →（一阶）人手变多 →（二阶）沟通成本暴涨，反而更慢（《人月神话》）。

技巧：任何决定，强迫自己往下追问**至少两层**「那会导致什么？再然后呢？」。

**练什么**：看见时间维度上的连锁反应，避免「解决一个问题、制造三个新问题」。

**本周任务**：挑一个你最近的决定或一条新闻政策，画一条「一阶 → 二阶 → 三阶」的后果链。
