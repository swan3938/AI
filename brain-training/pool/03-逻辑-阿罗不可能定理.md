# 🗳️ 逻辑 · 阿罗不可能定理：完美的投票制度，数学上不存在

我们总以为"民主投票"只是技术问题——只要规则设计得好，就能公平地把每个人的偏好汇总成集体选择。

1951 年，经济学家肯尼斯·阿罗用数学证明了一件让人不安的事：**当候选项有三个或以上时，不存在任何一种排序投票制度，能同时满足几条最basic的公平要求**：

- 不是独裁（结果不能只由一个人决定）；
- 大家都更喜欢 A 胜过 B，集体结果就该 A 胜 B（帕累托）；
- A 和 B 的相对排名，不该因为加进来一个无关的 C 而改变（无关选项独立性）。

这几条听起来天经地义，但阿罗证明：你**不可能全要**，必然有一条被违反。这不是"还没设计出好制度"，而是逻辑上根本做不到——他为此拿了诺奖。

它的冲击远超投票：任何"把多个个体偏好/多个指标，汇总成一个集体决定/综合排名"的场景，都受这个定理约束——公司决策、评分榜单、推荐算法的多目标排序……**"客观公正的综合排名"在数学上常常是个幻觉**。
